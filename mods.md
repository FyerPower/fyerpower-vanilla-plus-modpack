# Mod List

This document contains a comprehensive list of all mods included in the Forever - Reimagined modpack, organized by category.

---

## Big Mods
*Mods that add significant new content, blocks, or items to the game*

- **Lootr (Fabric)** - Makes dungeon and structure loot chests unique per player, preventing griefing and ensuring everyone gets their own loot.
- **Magnum Torch** - Adds a special torch that prevents mob spawning in a large radius around it.
- **Open Parties and Claims** - Comprehensive land claiming and party system for multiplayer servers, allowing players to protect their builds and team up.
- **Tom's Simple Storage Mod (Fabric)** - Adds a simple and intuitive storage network system with terminals, connectors, and inventory management.
- **Traveler's Backpack [Fabric]** - Adds wearable backpacks with built-in crafting table, fluid storage, and tool slots for portable storage.
- **Universal Graves** - Creates graves when players die, storing all items safely and preventing item loss.
- **Waystones** - Adds placeable waystone blocks that allow fast travel between discovered locations.
- **Xaero's Minimap** - Adds a highly customizable minimap to the game with waypoints, death points, and entity tracking.
- **Xaero's World Map** - Adds a full-screen world map that integrates with Xaero's Minimap for exploration tracking.

## Utility Mods
*Mods that provide players with extra utility and quality-of-life improvements*

- **AppleSkin** - Shows food saturation and hunger restoration values, helping players make informed eating decisions.
- **Better Advancements** - Improves the advancements screen with a larger, more readable interface and better navigation.
- **Better Statistics Screen** - Enhances the statistics screen with sorting, filtering, and improved organization.
- **BetterF3** - Makes the F3 debug screen more readable and customizable with colored text and formatting.
- **Chat Heads** - Displays player heads next to chat messages for easier identification of who's talking.
- **Chunky (Fabric)** - Pre-generates world chunks to reduce lag and improve performance during exploration.
- **Clumps** - Groups experience orbs together to reduce lag and make collection more efficient.
- **Durability Tooltip** - Shows item durability as a number in tooltips for precise tracking.
- **Enchantment Descriptions** - Adds helpful descriptions to enchantments explaining what they do.
- **Explorer's Compass** - Adds a compass that helps locate structures and biomes.
- **Item Scroller** - Adds convenient mouse controls for moving items in inventories quickly.

- **Just Enough Items (JEI)** - Recipe viewer that shows crafting recipes, uses, and item information.
- **Just Enough Professions (JEP)** - JEI addon showing villager professions and trades.
- **Just Enough Resources (JER)** - JEI addon showing world generation information, mob drops, and resource spawning.
- **Leash Villager** - Allows players to use leads on villagers for easier transportation.
- **Litematica** - Schematic mod for copying, pasting, and building structures with visual overlays.
- **MiniHUD** - Displays customizable on-screen information like coordinates, light levels, and biome data.
- **Mod Menu - Community Updated** - Adds a mod list screen with configuration options for installed mods.

- **Status Effect Bars** - Shows compact status effect bars above the hotbar for better visibility.
- **Styled Chat** - Enhances chat with formatting, markdown support, and better readability.
- **Styled Nicknames** - Allows players to set custom nicknames with formatting.
- **Styled Player List** - Improves the player list with formatting and customization options.

## Performance Mods
*Mods that optimize game performance and improve FPS*

- **Concurrent Chunk Management Engine** - Improves chunk loading and generation performance through multi-threading.
- **Cull Leaves** - Optimizes leaf rendering by culling hidden faces, improving FPS in forests.
- **Dynamic FPS** - Reduces FPS when Minecraft is running in the background to save resources.
- **Entity Culling Fabric/Forge** - Skips rendering entities that are behind walls or blocks, improving performance.
- **ImmediatelyFast** - Optimizes various rendering operations for better FPS and reduced lag.
- **Iris Shaders** - Adds shader support with better performance than OptiFine, compatible with Sodium.
- **LambDynamicLights** - Adds dynamic lighting from held items and entities without impacting performance.
- **Leaves Us In Peace** - Prevents leaf decay particles and updates, improving performance near trees.
- **Let Me Despawn** - Allows passive mobs to despawn naturally, reducing entity lag.
- **Lithium (Fabric/NeoForge)** - General-purpose optimization mod improving many game systems without changing behavior.
- **MoreCulling** - Additional culling optimizations for leaves, particles, and other elements.
- **ServerCore** - Server-side optimization mod improving server performance and reducing lag.
- **Sodium Extra** - Adds extra options and features to Sodium for further customization.
- **Sodium Shadowy Path Blocks** - Fixes path block rendering issues when using Sodium with shaders.
- **Sodium** - Complete rendering engine replacement dramatically improving FPS and performance.
- **Sound Physics Remastered** - Adds realistic sound physics (reverb, attenuation) with optimized performance.
- **Very Many Players (Fabric)** - Optimizes servers to handle many players simultaneously without lag.
- **spark** - Performance profiler for diagnosing lag and performance issues on clients and servers.

## Library Mods
*Dependency mods that provide APIs and frameworks for other mods*

- **Almanac Lib** - Library mod providing shared functionality for other mods.
- **Architectury API** - Cross-platform API allowing mods to work on multiple mod loaders.
- **Balm** - Multi-loader library providing common utilities for mod development.
- **Cardinal Components API** - API for attaching data to various game objects in Fabric.
- **Cloth Config API (Fabric/Forge/NeoForge)** - Configuration screen library used by many mods.
- **Fabric API** - Core API mod required for most Fabric mods to function.
- **Fabric Language Kotlin** - Allows mods to be written in Kotlin programming language.
- **Forge Config API Port** - Ports Forge's configuration system to Fabric.
- **MaLiLib** - Library mod required for various masa's mods (Litematica, MiniHUD, Tweakeroo).
- **MidnightLib** - Configuration library used by various mods.
- **Polymer** - Server-side library for creating custom content without requiring clients to install mods.
- **Prickle** - Library providing shared functionality for mod development.
- **Puzzles Lib** - Library mod providing common utilities and APIs.
- **SuperMartijn642's Config Lib** - Configuration library used by SuperMartijn642's mods.
- **TCDCommons API** - Common library for TheCyberBrick's mods.
- **Text Placeholder API** - API for text placeholders used by styled mods.
